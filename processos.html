Para a concretização do projeto o grupo, elaborou previamente uma pesquisa profunda de forma a encontrar uma opinião geral sobre o Projeto Europeu. Apesar das opiniões favoráveis, tornou-se notório um sentimento de incerteza, de indiferença, de insegurança na sua capacidade de responder às espectativas que lhe são atribuídas por parte dos seus cidadãos.
A União Europeia é uma união de identidades distintas onde é importante colocar a questão: de que forma é que nos podemos unir para que este projeto seja aquilo que idealmente pretendia ser? A cultura poderá ser a resposta, um meio para criar um espaço de união, uma alternativa ao conflito.
Para chegar a esta abordagem, em que a cultura poderá ser o meio para criar uma europa unida, os membros do grupo aprofundaram, paralelamente, a sua perceção da União Europeia, algumas das questões que os preocupam e qual é o seu verdadeiro significado. Explore esses processos aqui, de forma a compreender a origem do projeto ‘ÁGORA'.
